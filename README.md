## Content

- iterator
- _type_traits
- uninitialized
- stl_vector
- main: Test stl_vector.

## Usage
- Install gtest library.
- Copy the project into C++ IDE
  - Run main.cpp.

- Or use CMakeLists.txt
  ```
  cd build
  cmake ..
  make
  ./demo
  ```
- Test OK in Mac M1. Big Sur 11.3.1.
